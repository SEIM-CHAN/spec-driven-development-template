# 見積基盤

このフォルダでは、作業工数と金額の根拠を、要求・要件・変更要求と結び付けて管理します。見積は約束ではなく、特定時点のスコープ・前提・リスクに基づく判断です。

- 見積方針と単価の扱い: `ESTIMATION_POLICY.md`
- 見積書の雛形: `EST-001-template.md`
- 内部計算ブック: `internal_estimate_calculator.xlsx`（顧客共有禁止）
- 顧客提示用ブック: `client_estimate_share_template.xlsx`
- 顧客提示用への出力手順: `CLIENT_EXPORT.md`
- 変更による増減: `CR-xxx` と対応する `EST-xxx` の差分で管理

エージェントは作業分解、過去実績の集計、抜け漏れ、変更差分の整理を支援できます。ただし、単価、生産性係数、予備費率、最終金額の承認を推測・確定してはいけません。
